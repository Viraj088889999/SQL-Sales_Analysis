# SQL-Sales_Analysis
SQL project analyzing pizza sales using MySQL to generate business insights through joins, aggregations, and advanced SQL queries.


# 🍕 Pizza Sales Analysis using SQL:

A complete SQL data analysis project that explores a real-world pizza sales dataset using MySQL to answer business questions, generate actionable insights, and demonstrate SQL skills ranging from basic queries to advanced analytical techniques.

---

# Project Overview:

This project analyzes historical pizza sales data using SQL to uncover valuable business insights. By querying multiple related tables, the project identifies sales trends, customer ordering behavior, product performance, and revenue distribution.

The analysis demonstrates how SQL can be used to solve real business problems through data exploration, aggregation, joins, subqueries, and window functions.

# Business Problem:

Restaurants generate large amounts of sales data every day. However, without proper analysis, it is difficult to answer important business questions such as:

- Which pizzas generate the highest revenue?
- What are the busiest ordering hours?
- Which pizza sizes are most popular?
- Which categories contribute the most to sales?
- How does revenue grow over time?

This project answers these questions using SQL and converts raw transactional data into meaningful business insights.

# Dataset Information:

The project uses a Pizza Sales relational database consisting of four interconnected tables.

## Tables:

| Table | Description |
|--------|-------------|
| orders | Stores customer order information |
| order_details | Stores individual pizzas ordered in each order |
| pizzas | Stores pizza prices and sizes |
| pizza_types | Stores pizza names, ingredients, and categories |

# Business Questions Solved:

## Basic Analysis:

- Retrieve the total number of orders placed.
- Calculate total revenue generated.
- Identify the highest-priced pizza.
- Find the most commonly ordered pizza size.
- List the top 5 most ordered pizza types.

---

## Intermediate Analysis:

- Calculate total quantity sold by pizza category.
- Analyze order distribution by hour.
- Find category-wise pizza distribution.
- Calculate the average number of pizzas ordered per day.
- Determine the top 3 pizza types based on revenue.

---

## Advanced Analysis:

- Calculate percentage contribution of each category to total revenue.
- Analyze cumulative revenue over time.
- Determine the top 3 revenue-generating pizzas for each category.

---

# Key Business Insights:

Some important findings from the analysis include:

- Large-sized pizzas were ordered most frequently.
- The Thai Chicken Pizza generated the highest revenue.
- Peak ordering hours occurred during the evening.
- Chicken and Classic pizzas contributed significantly to total revenue.
- Revenue increased steadily throughout the sales period.
